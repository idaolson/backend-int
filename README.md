## Local Set-up:
**Version Requirements**
* Ruby 2.7.2
* Rails 5.2.6

1. `git clone git@github.com:idaolson/backend-int.git`
2. `cd backend-int`
3. `bundle install`
4. `rails db:{create,migrate}`
5. `rails db:seed`
6.  run: `rails s` and explore the endpoints below
7.  run the test suite: `bundle exec rspec`
